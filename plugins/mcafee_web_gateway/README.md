# McAffee Web Gateway checks

Only infomational (always returns 0 - OK) checks for:

 - Infections Detected
 - Connections Allowed
 - Connections Blocked by Anti-Malware
 - Connections Blocked by Media-Type Filter
 - HTTP requests
 - HTTPS requests
 - Clients Connected
